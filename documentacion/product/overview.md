# Product Overview

## Что это сейчас

Пока весь портал UISamurai = UI-тренажер.

## Главный экран

- Маршрут `/` и до логина, и после логина показывает UI-тренажер.
- Верх: логотип + кнопка входа.
- Hero: заголовок и описание тренажера.
- Ниже: слева навигация по разделам, справа контент раздела.
- Отдельного титульного экрана нет.

## Доступы

- Основная теория/контент доступна без авторизации.
- Блок задания в конце раздела доступен только после логина.
- Комментарии доступны только после логина.

## Навигация

- По умолчанию открывается первый опубликованный раздел из сайдбара.
