# UI Trainer

## Layout

- Главная страница портала = UI-тренажер.
- Слева: список разделов (сайдбар).
- Справа: контент активного раздела.
- Вверху: логотип и кнопка входа.

## Поведение

- При открытии `/` сразу показывается первый опубликованный раздел.
- Отдельного титульного экрана нет.
- Черновики не показываются публично.

## Контентная модель

- `sections`: slug, title, sort_order, is_published.
- `body`: публичная теория (доступна всем).
- `assignment`: блок задания (только после логина).

## Админский контент

Контент создается и редактируется через админку:

- заголовки/подзаголовки
- текст
- сноски
- изображения (через Supabase Storage)
