# Admin

## Роль

В v1 админ только один (Арман).

## Доступ

- Маршруты `/admin/*`: без сессии Telegram показывается подсказка «войди на главной»; при входе без `is_admin` — «нет прав» (не HTTP 404).
- Проверка по серверной сессии + `users.is_admin`.

## Что делает админ

- Создает разделы.
- Меняет порядок разделов.
- Редактирует `body` и `assignment`.
- Загружает изображения для контента.
- Публикует/снимает с публикации раздел.

## Черновики

- Видны только в админке.
- Публичная часть показывает только опубликованные разделы.

## Комментарии в админке

- Админ видит все комментарии по всем разделам.
- Новые комментарии отображаются в общем списке, чтобы не пропускать обсуждения.
- Админ может отвечать на комментарии от своего аккаунта.
