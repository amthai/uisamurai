# UISamurai

Портал для дизайнеров и айтишников. Текущий фокус первой версии: UI-тренажер.

## Tech

- Next.js (App Router) + TypeScript
- Supabase Postgres + Storage
- Telegram Login (без Supabase Auth)

## Local Run

1. Установить зависимости:

```bash
npm install
```

2. Создать `.env.local` по шаблону из `.env.example`.
3. Запустить dev:

```bash
npm run dev
```
